Victor Aldecoa website
======================

Develop
-------
```bash
clj -M -m cljs.main -d "victoraldecoa.github.io/out" -c victoraldecoa.static-website -r
```

Compile
-------
```bash
clj -m cljs.main -O advanced -c "demo.static-website"
```

--------
